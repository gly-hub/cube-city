# 关卡最优值重新计算（考虑道路占用）

## 关键约束

1. **道路需求**：
   - 除了 `wind_power`（风力发电）和 `park`（公园），所有建筑都需要相邻有道路
   - 道路本身也占用格子

2. **道路布局**：
   - 最优布局是网格状道路系统
   - 道路占用约 15-20% 的格子（网格布局）
   - 采用 18% 作为保守估计

3. **建筑分布**：
   - 需要道路的建筑：工业、住宅、商业、大部分环境设施、基础设施
   - 不需要道路的建筑：风能发电、公园（可以放在任意位置）
