# 2.5D 卡通城市放置系统 (CubeCity)

> 基于 Three.js 和 Vue 的轻量级 2.5D 城市建造模拟游戏

欢迎来到 CubeCity！这是一款卡通风格的 2.5D 城市模拟游戏，你可以建造、管理和扩展属于你自己的大都市。放置建筑、铺设道路，在管理资源和扩展领地的过程中见证你的城市成长。

![Gameplay Demo](README/游玩时动图.gif)

## ✨ 核心功能

*   **🏙️ 自由建设:** 随心所欲地放置、移动和拆除各类建筑与道路，打造独一无二的城市景观。
*   **🧩 策略规划:** 平衡住宅 (R)、商业 (C)、工业 (I) 的发展，同时兼顾环境 (E)、社会 (S)、治理 (G) 的需求，实现城市可持续发展。
*   **💰 经济系统:** 建筑会自动产出金币，利用这些金币来建造新建筑、升级或扩展你的领地。
*   **🔬 科技树系统:** 建筑升级到最高级后可研发科技，提升建筑性能和城市效率。
*   **📊 系统状态:** 动态计算电网、交通、安全、环境等系统状态，影响城市收入。
*   **💾 本地存储:** 你的城市进度会自动保存在本地，随时可以回来继续你的建设大业。
*   **🎨 卡通风味:** 明快的色彩和可爱的卡通模型，带来轻松愉悦的视觉体验。

## 🎮 玩法介绍

游戏主要围绕四种操作模式展开，让你轻松管理城市的方方面面：

*   **🔍 选择模式 (SELECT):**
    *   点击建筑查看详细信息，如居民数量、状态、产出等。
    *   满足条件时可对建筑进行升级，提升其功能和产出。
    *   3级建筑可研发科技树，进一步提升性能。

*   **🏗️ 建造模式 (BUILD):**
    *   从左侧面板选择你想要的建筑。
    *   在地图上的可用地皮上点击即可放置建筑，实时预览模型和高亮提示让操作更直观。

*   **🚚 搬迁模式 (RELOCATE):**
    *   选中一个已建好的建筑，然后点击一个空地，即可轻松完成搬迁。
    *   在放置前，可以旋转建筑以适应你的城市布局。

*   **💣 拆除模式 (DEMOLISH):**
    *   切换到此模式，点击不再需要的建筑即可将其拆除。
    *   拆除建筑会返还部分建造成本。

## 🚀 快速开始

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### Docker 部署

```bash
docker run -d \
  --name cube-city \
  --restart always \
  -p 5141:5141 \
  cube-city:latest
```
